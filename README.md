# Project-1 Login Automation Framework

## Overview

This project automates the Login functionality of SauceDemo using Selenium WebDriver, Java, TestNG, and the Page Object Model (POM) design pattern.

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Git
- GitHub

## Framework Structure

- BaseTest
- CommonUtils
- LoginPage
- LoginTest

## Test Scenarios Covered

1. Valid Login

2. Invalid Username

3. Invalid Password

4. Invalid Username and Password

5. Empty Username

6. Empty Password

7. Empty Credentials

## Application Under Test

https://www.saucedemo.com/

## Design Pattern

Page Object Model (POM)
